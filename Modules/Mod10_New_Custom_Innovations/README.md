# Module 10 — New custom innovations

Course module for **new custom work**: remote control, logging, and related integrations.

| Subfolder | Contents |
|-----------|----------|
| **`RemoteControlCart/`** | RC cart — CAD references, photos, spreadsheet; **see [`RemoteControlCart/README.md`](RemoteControlCart/README.md)** for links to **`CADfiles`** RC enclosure STLs and **`ArduinoCodes/RcController`**. |
| **`BleAppDataLogger/`** | Android app APK distribution + tester guide for BLE logging with **`ArduinoCodes/SDLCDUnoR4_BLE_power_logger`**. |
| **`SDcardLogger/`** | SD / logging–related assets for this module. |

**RC electronics printed parts** (`RC_Electronics_Case_v1.0.stl`, `RC_Electronics_Lid_v1.0.stl`) live in **`CADfiles/`** and are indexed in **[`CADfiles/README.md`](../../CADfiles/README.md)**.
